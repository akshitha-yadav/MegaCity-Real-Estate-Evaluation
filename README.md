# 🏙️ MegaCity Real Estate Evaluation

A Django-based web application designed to streamline property management and real estate evaluation. The platform enables users to browse properties, view details, and manage real estate information through an intuitive interface.

## 🚀 Features

- User Registration & Login
- Property Listings Management
- Property Search & Filtering
- Property Evaluation System
- Admin Dashboard
- Responsive User Interface
- Secure Database Management
- Image Upload & Media Management

## 🛠️ Tech Stack

- Python
- Django
- SQLite3
- HTML5
- CSS3
- JavaScript
- Bootstrap

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/akshitha-yadav/MegaCity-Real-Estate-Evaluation.git
cd MegaCity-Real-Estate-Evaluation
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/Mac**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
